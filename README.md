# Bump

Bump version info in a repository

## Installation

Add this line to your application's Gemfile:

    gem 'bump'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bump

## Usage

```
bump [-s|-m|-l] [-f]
```
